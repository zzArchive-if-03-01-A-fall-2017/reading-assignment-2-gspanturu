Compoments of html elements

start tag - end tag + content in between

the html element < html /> defines the whole document.

h1, h2, h3, h4, h5, h6 - heading

Void elements

Void elements: elements with no content

Example: <br/>

Attributes

All HTML elements can have attributes

Attributes provide additional information about an element

They're always specified in the start tag

They usually come in name/value pairs: name="value"

Attributes
hrefimage width and height style lang title
< a href = "http://facebook.com"> This is a fb link </a/> < img src="image.jpg"/> < img src="image.jpg" width="500" height="600"/> <p style="color:red">I am a paragraph </p> < html lang="en-US" > (added to the < p > element, displayed when the mouse is over the p)
alt(alternative text, if image can't be desplayed): < img src="image.jpg" alt="An image"/> < p title="I'm a tooltip"> This is a paragraph </p>

Fonts

Serif: fonts have small lines at the ends on some characters

sans-serif: sans=without, no lines

font-family: should hold some font names as an option if the browser doesn't support the 1st font

Example:

p {

font-family: "Times New Roman", Times, serif;

}

Font Style:

p.normal/italic{

font-style: normal/italic;

Font Size

-absolute size:

  • sets the text to a specified size
  • user can't change the text size
  • useful when the size of the output is known

font-size: small;

-relative size:

  • size set relative to surrounding elements
  • user can change the text size

font-size: smaller;

Font Weight

p.normal/thick {

font-weight: normal/bold;

}
